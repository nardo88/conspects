Код CSS


ul li {
border: 1px solid silver;
display: inline-block;
}

ul ul {
display: none;
position: absolute;
padding: 0;
margin: 0;
background-color: #00bfff;
}

li:hover ul {
display: block;
}

li:hover li {
display: block;
}
a:hover {
color: #00008b;
}

a {
font-size: 1.5em;
color: #fff;
text-decoration: none;
}

li:hover {
background-color: #fff;
}

ul {
display: inline-block;
background-color: #ff7f24;
padding: 0;
margin: 10px;
}
Комментарии

1 Блок ul li (стр 1-4)

  • В строке 2 мы указываем рамку толщиной 1 пиксель, она сплошная и цвет silver
  • В строке 3 указываем display для li которые находятся в ul, т.е. слова Menu и Settings выстроятся в линию и станут блоками

2 Блок ul ul (стр 6-12)

  • В строке 7 мы скрываем все ul которые находятся в родительской ul.
  • В строке 8 дочерние ul принимают позицию absolute
  • В строках 9 и 10 мы указываем отступы с внешней и внутренней стороны
  • В строке 11 мы указываем фоновый цвет ячеек выпадающего меню (слова Close, New и т.д.)

3 Блок li:hover ul (стр 14-16)

  • В этом блоке мы говорим, что все дочерние ul которые находятся в родительских li будут блоками при наведении мыши. Но в таком случае выпадающие списки будут выстраиваться в линию, а не в столбец. Для того что бы меню выстраивалось в столбец пишем следующий блок

4 Блок li:hover li (стр 18-20)

  • Все дочерние li находящиеся в родительских дш при наведении станут блоко. Т.е. 3 блок позволяет при наведении выпадать дочерний список, а 4 блок выравнивает его в столбец

5 Блок a:hover(стр 21-23)

  • При наведении мышки на ссылку цвет текста будет меняться на #00008B (синий)

6 Блок a (стр 25-29)

  • Строка 26 устанавливает размер текста ссылки 1.5em
  • Строка 27 устанавливает цвет текста по умолчанию #fff (белый)
  • 28 строка убирает подчеркивание у ссылки

7 Блок li:hover (стр 31-33)

  • В строке 32 указывается, что при наведении на блоки li фоновый цвет будет меняться на белый (#fff)

8 Блок ul (стр 35-40)

  • 36 строка делает родительские ul блоками
  • 37 строка устанавливает цвет фона родительским ul
  • 38 строка устанавливает отступ внутри блока 0px
  • 39 строка устанавливает внешний отступ 10px
