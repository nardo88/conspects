Подготовка

Для корректной работы слайдера необходимо подключить три библиотеки:

1. Библиотеку Jquery

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>

2. Файл CSS для слайдера

<link href="https://cdnjs.cloudflare.com/ajax/libs/fotorama/4.6.4/fotorama.css" rel="stylesheet">

3. Файл java Script для слайдера

<script src="https://cdnjs.cloudflare.com/ajax/libs/fotorama/4.6.4/fotorama.js"></script>

Затем в блок предназначенный для слайдера добавляем код самого слайдера

<div class="fotorama"> <img src="https://s.fotorama.io/1.jpg"> <img src="https://s.fotorama.io/2.jpg"> </div>

Внутри блока с классом fotorama помещаются теги img с сылками на картинки слайдера

Так же можно подключить не через CDN. Для этого скачиваем архим с официального сайта, в архиве будет два файла библиотеки CSS и JavaScript. Ну а Jquery просто скачиваем с сайта jquery.com

Странно то, что если подключать не через CDN а физически, то некоторые вещи не работают. Так что лучше использовать CDN ссылки
