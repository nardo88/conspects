<html> <head> <meta charset="utf-8"> <script type="text/javascript"> function MyFunction () { var selind = document.getElementById('MyFirstSelect').options.selectedIndex; var text = document.getElementById("MyFirstSelect").options[selind].text; if (text == "Кузнецов М.А.") { MyForm.uchetka.value = "4849_KuznecovMA"; MyForm.pass.value = "ufk48_MA"; } if (text == "Гиматова Д.А.") { MyForm.uchetka.value = "4849_GimatovaDA"; MyForm.pass.value = "ufk48_DA"; } } </script> </head> <body> <form name="MyForm"> <select id="MyFirstSelect"> <option>Кузнецов М.А.</option> <option>Гиматова Д.А.</option> </select> <input type="button" value="Получить данные" name="" onclick="MyFunction()"> <input type="text" name="uchetka"> <input type="text" name="pass"> </form> </body> </html>

Рассмотрим код

Мы имеем в теле нашего html файла форму, которой мы дали имя - "MyForm" (стр 26-37). В этой форме находятся четыре атрибута:

  • выпадающий список select с присвоенным ему ID "MyFirstSelect" (стр 28-31)
  • кнопка input типа button. Именно эта кнопка и будет содержать событие onclick, которое будет вызывать функцию "MyFunction" (стр 33)
  • поле ввода input с именем "uchetka"
  • поле ввода input с именем "pass"

Цель

при выборе атрибута выпадающего списка и после нажатия книпки поля ввода должны заполнится данными учетных записей выбранного пользователя

  • В начале нам необходимо объявить две переменные. Первая переменная selind (стр 6), ей будет присваиваться индекс выбранного в данный момент поля из выпадающего списка. Присваивание происходит с помощью метода "getElementById". Входящим параметром мы указываем ID нашего select. Далее мы выбираем метод options который содержит функцию selectedIndex
  • Второй переменной (стр 7) присваивается значение атрибута списка. Присваивание происходит так же через метод "getElementById", только в этом случае у метода options есть входящий параметр, который мы указываем в квадратных скобках. в качестве параметра надо указать индекс нужной нам строки списка, а так как переменной selind мы присвоили именно этот индекс, то в качестве входящего параметра метода options мы указываем переменную selind
  • Ну а дальше идет оператор сравнения: если text равен "Кузнецов М.А." то записать в MyForm.uchetka.value такой-то текст.
