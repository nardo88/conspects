Создаем две переменные - включающие в себя маасив элементов блоков навигации с табами и контентом табов

Далее запускаем цикл в котором отслеживаем событие Click у всех элементов с классом tab_nav_item (это блоки с навигацией). При собитии Click будет вызываться функция addActive

Далее тело самой функции. Начинается она с того что запускается цикл в котором у всех элементов удаляется дополнительные класс который делает блок видимым и применяет стили к активному блоку

Далее мы создаем переменную, в которую помещаем текущий элемент, доступ к которому мы получаем с помощью метода target события event

Затем запускается второй цикл, в котором мы перебираем все навигационные табы и если переменная tsrget равна элементу цикла с текущим индексом (т.е. так мы отловили индекс элемента по которому был совершен клик) то элементам двух массивов с этим же индексом добавляются дополнительные активные классы.

Код JavaScript

const tabNavItem = document.querySelectorAll('.tab_nav_item'); const tabContentItem = document.querySelectorAll('.tab_content_item'); for ( var ind = 0; ind < tabNavItem.length; ind++) { tabNavItem[ind].addEventListener('click', addActive); } function addActive(event) { for ( var ind = 0; ind < tabNavItem.length; ind++) { tabNavItem[ind].classList.remove("active_tab"); tabContentItem[ind].classList.remove("active_tab_contant"); } var target = event.target; for (var i = 0; i < tabNavItem.length; i++) { if (target == tabNavItem[i]){ tabNavItem[i].classList.add("active_tab"); tabContentItem[i].classList.add("active_tab_contant"); } } }
