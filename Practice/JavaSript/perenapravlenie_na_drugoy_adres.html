Команда перенаправления на другой сайт с помощью JavaScript

1. Создаем кнопку с событием onclikc

<input type="button" onclick="Myfunction()">

2. Создаем функцию с кодом:

function Myfunction() { window.location = "http://google.com"; }

Если указать window.reload, то по нажатии на кнопку наша страница обновится
