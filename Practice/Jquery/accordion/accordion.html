Аккордеон

  • HTML

    Здесь мы испозьзуем список ul. Каждая строка списка будет содержать два блока. Первый - это заголовок а второй - это содержание. При нажатии на заголовок мы будем отображать или скрывать блок с содержанием при помощи функции JQuery slideUp или slideDown

  • CSS

    .accordion - задаем стили списку

    • list-style-type: none; - убираем кружки
    • border-top: 1px solid #ddd; - стилизуем верхнюю границу рамки
    • border-left: 1px solid #ddd; - стилизуем левую границу рамки
    • border-right: 1px solid #ddd; - стилизуем правую границу рамки
    • margin: 0 auto; - выравниваем наш список по центру
    • padding-left: 0; - внутренний отступ слева - 0
    • width: 95%; - ширина списка 95% от ширины экрана

    .accordion-header - стилизуем блок который содержит заголовок

    • background-color: #fff; - выставляем цвет фона
    • border-bottom: 1px solid #ddd; - стилизуем рамку
    • cursor: pointer; - меняем курсор при наведении
    • min-height: 3rem; - минимальная высота блока
    • line-height: 3rem; - высота строки
    • padding: 0 1rem; - внутренние отступы сверху снизу - по бокам 1rem

    .accordion-header:after - здесь мы применяем к блоку с заголовком псевдокласс after. Этот псевдокласс добавляет после содержимого элемента необходимый текст. передаваемый текст помещается в свойство content. Так же в этом разделе CSS мы стилизуем именно этот текст который будет добавлен.

    • content: '\02795'; - \02795 - это код крестика
    • font-size: 0.6rem; - выставляем размер крестику
    • color: #777; - выставляем цвет креста
    • float: right; - и говорим что выравнивает его следует по правой стороне

    .accordion-header.active:after - эта строка читается так: элементу с классом .accordion-header.active применяется эффект псевдокласса after. т.е. с помощью JQuery мы будем элементу с классом .accordion-header добавлять дополнительный класс и тогда элементу будут применяться те стили которые мы указали в этом разделе

    • content: '\02796'; - \02796 - это код минуса.

    .accordion-body - стилизуем блок с содержимым статьи

    • display: none; - отключаем блок
    • padding: 1rem; - внутренний отступ
    • border-bottom: 1px solid #777; - стилизуем нижнюю часть рамки

  • JavaScript

    Использую JQuery мы обращаемся к элементу с классом accordion-header и обрабатываем на нем собитые click. Клик будет вызывать функцию, которая применяет к элементу с классом accordion-header метод toggleClass (этот метод добавляет в случае отсутствия класса active у элемента или убирает в случае наличия класса active).

    На следующей строке мы так же обращаемся к элементу с классом accordion-header и применяем ему метод next (этот метод применяет последующее действие к элементу который идет следом,в нашем случае это блок с содержанием) и затем к этому блоку применяем тот же метод добавления класса active.

    После этого так же мы применяем к следующему элементу (это элемент - блок с содержанием) метод slideUp. В нашем случае slideUp будет скрывать блок с содержанием

    В конце кода мы используем оператор if. Если существует элемент с классом '.accordion-body.active', то к этому элементу применяем метод slideDown, тоесть отображать его
