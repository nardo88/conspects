Контейнерные компоненты (Container component)

Есть два вида компонент

  • Презентационная
  • Контейнерная

Презентационная компонента должна заниматься только тем, что отрисовывать JSX код. Эта компонента не должна значть о логике в store, она должна принимать в props фиксированные значения и передавать через CallBack функции значения которые получает из UI.

Контейнерная компонента наоборот имеет полный доступ к store и обеспечивает презентационную компоненту данными из store. Контейнерная компонента она отрисовывает Презентационную передавая ей через props только необходимые данные.
