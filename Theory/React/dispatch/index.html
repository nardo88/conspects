Dispatch

dispatch - это один единственный метод в state (объекте), который будет изменять любую ветку в state. Т.е. мы будем прокидывать через props всего один метод dispatch и он будет менять ту ветку в state, которую нам нужно

dispatch должен принимать в качестве аргумента объект action который будет содержать описание нужного дейтсвия и принимаемые данные. Каким образом будет происходить описание нужного действия? action должен иметь свойство - type - это обыкновенная строка с описанием действия. Вот пример:

Как правило dispatch мы будем должны пробросить через props до компоненты. При передаче dispatch не забываем забайндить его что бы не потерять контекст вызова

Теперь все что нам остается, это протащить через props-ы dispatch и вызвать его в функции которая будет вызываться по нужному событию (например при нажатии на кнопку).
