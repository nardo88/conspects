Атрибуты

Элементам DOM соответствуют HTML-теги, у которых есть текстовые атрибуты.

Доступ к атрибутам осуществляется при помощи стандартных методов:

  • elem.hasAttribute(name) - проверяет наличие атрибута
  • elem.getAttribute(name) - получает значение атрибута
  • elem.setAttribute(name, value) - устанавливает атрибут
  • elem.removeAttribute(name) - удаляет атрибут

Также все атрибуты элемента можно получить с помощью свойства elem.attributes

Обращение к свойствам:

В отличие от свойств, атрибуты:

  • Всегда являются строками.
  • Их имя нечувствительно к регистру (ведь это HTML)
  • Видны в innerHTML (за исключением старых IE)

свойство value - elem.value

свойство classList - elem.classList

Методы classList:

  • elem.classList.contains("class") - возвращает true/false, в зависимости от того, есть ли у элемента класс class
  • elem.classList.add/remove("class") - добавляет/удаляет класс class
  • elem.classList.toggle("class") - если класса class нет, добавляет его, если есть – удаляет.

Обращение к свойствам CSS

elem.style.styleName

Пример. Как получить значение свойства display

var s = elem.style.display;

Пример как задать свойство CSS

elem.style.display = 'block';

Некоторые атрибуты элемента:

  • elem.offsetHeight - высота элемента
  • elem.offsetWidth - ширина элемента
