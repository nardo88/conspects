addEventListener()

Метод addEventListener() регистрирует определенный обработчик события, вызванного на определенный элемент.

Синтаксис

target.addEventListener('event', function);

  • target - объект к которому применяется метод addEventListener
  • event - событие которое метод будет прослушивать
  • function - фунция которая будет выполнена в случае события
