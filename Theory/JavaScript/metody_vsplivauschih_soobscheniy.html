Методы всплывающих сообщений

  1. метод Alert

Этот метод вызывает всплывающее окно

Пример:

var str = “Max”;

Alert (“Hello ” + str);

В итоге откроется окно с надписью «Hello Max»

  1. метод confirm

Этот метод вызывает всплывающее окно с двумя кнопками: «ОК» и «Cancel». У метода есть возвращаемый параметр типа Boolean. Если жмем ОК – результат True, Cancel – False.

Пример:

If (confirm (“Are you sure”)){

Alert (“Hello”);

}

Else {

Alert (“Good Bay”);

}

  1. метод prompt

Этот метод вызывает окно с полем ввода и то что мы введем присваивается выходному параметру метода. Т.е. значение выходного параметра мы можем присвоить любой другой переменной.

Пример

var cop;

cop = prompt (“введите свое имя”);

alert (“ваше имя ”+ cop);
