Чтение и запись data атрибутов

Метод data() - читает или устанавливает данные связанные с любым элементом документа. Методу data() передается два аргумента: имя и значение

Если передать методу data() объект, то свойства этого объекта заменят все ранее связанные данные

При вызове метода без аргументов он вернет объект с уже имеющимися данными.

Для удаления данных используется метод removeData() в качестве аргумента передается строка с именем данных. Если вызвать буз аргументов то будут удалены все данные.
