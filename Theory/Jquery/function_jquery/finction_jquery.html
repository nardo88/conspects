Когда функции $() передается CSS селектор, она возвращает объект представляющий множество элементов (псевдомассив)

Длина массива

$('div').length

либо

$('div').size()

Обращение к определенному элементу

$('div')[0]

либо

$('p').get(0)

Что бы преобразовать объект в настоящий массив

$('p').toArray()
