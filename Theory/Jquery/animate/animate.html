Анимационные эффекты

Простые эффекты

fadeIn() - элемент появляется (управление свойством opaciti)

fadeOut() - элемент исчезает (управление свойством opaciti)

fadeTo() - определяет конечное значение прозрачности и плавно изменяет текущее значение. принимает в себя несколько аргументов (1 - скорость анимациив милисекундах, 2 - значение opaciti)

show() - показывает элемент (управление свойствами width, height и затем display) в качестве аргумента можно передать скорость анимации в милисекундах

hide() - скрывает элемент (управление свойствами width, height и затем display) в качестве аргумента можно передать скорость анимации в милисекундах

toggle() - скрывает если виден и показывает если скрыт (управление свойствами width, height и затем display) в качестве аргумента можно передать скорость анимации в милисекундах

slideUp() - скрывает выбранный элемент уменьшая его высоту до 0 и затем устанавливает свойство display: none;

slideDown() - отображает элемент - противоположен методу slideUp()

slideToggle() - отображает если скрыт и скрывает если отображен

Реализация совственных эффектов

Метод animate() позволяет воспроизводить более сложные анимации. В качестве первого аргумента передается объект со значениями CSS их конечных значений. Во втором (необязательном) аргументе можно передать объект с параметрами эффекта:

  • duration - продолжительность анимации в милисекундах
  • complete - в это свойство можно поместить функцию, которая будет выполнена после окончания анимации

Что бы остановить анимацию нужно применить метод stop()
