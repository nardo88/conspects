Работа с классами

addClass() - добавляет класс

removeClass() - удаляет класс

toggleClass() - добавляет класс если он отсутствует и удаляет если он уже есть

hasClass() - проверяет наличие класса (возвращает булевое значение)

Получение значений элемента DOM

val() - получает значение value у элементов формы (input, select, textarea)

text() - читает и записывает содержимое элемента в виде простого текста

Lorem ipsum dolor sit amet.

Нажми на меня

html() - читает и записывает содержимое элемента в виде html разметки

Чтение и запись геометрии элемента

Метод offset() - позволяет получить или изменить координаты элемента. (если вызвать без аргументов, то вернет объект с координатами по X и Y). Коррдинаты относительно начала документа.

Метод position() - похож на offset за исключением что он позволяет только читать координаты не относительно начала документа, а относительно родителя.

Методы width() и height() возвращают ширину и высоту элемента не включающие ширину обводки и размеры отступов

Методы innerWidth() и innerHeight() - возвращают ширину и высоту с внутренними отступами

Методы scrollTop() и scrollLEft() - возвращают позицию полос прокрутки до элемента по осям X и Y

append() - добавляет в конец элемента
