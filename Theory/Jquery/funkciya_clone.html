Функция clone()

Эта функция позволяет клонировать объекты

Пример:

<div id = “div”> текст </div>

<script>

var block = $(“#div”).clone();

block.html(“<p> Новый абзац </p>”);

block.css(“font-size”, “40px”);

$(“body”).append(block);

</script>



И так, у нас есть блок, который заключен в div. В скрипте мы создаем переменную и записываем туда клон нашего блока (делается это с помощью метода clone()).

Далее мы редактируем состав и стиль нашел блока, который хранится в переменной.

И в конце мы с помощью $ обращаемся к объекту body и помещаем в него наш блок из переменной block используя метод append.



Append – этот метод помещает в объекты то, что мы укажем входящим параметром.

Пример:

$(“div”).append(“<span> Простой текст </span>”);



Remove – этот метод удаляет элемент

Пример:

$(“div”).remove();



empty – этот метод очищает элемент

Пример:

$(“div”).empty();
