Просмотр значений атрибутов объектов

Рассмотрим следующий код

var elements = $("img");

var text = "";

for (ver i = 0; i < elements.length; i++) {

text += elements[i].alt + "\n";

}

alert (text);

Переменной element присвоили значение всех объектов img, после чего element стал массивом.

Далее запускаем цикл for: от переменной i = 0 до i меньше длины массива (elements.length), переменной text присваивается значения атрибутов всех ячеек массива

"\n" - так как информация выводится методом alert, то эта строчка переносит текст на новую строку каждый тик цикла
