Добавление атрибута объекту

Рассмотрим следующий код:

$(":not[alt]").attr("alt", "изображение");

Эта строчка говорит о том, что для всех объектов img у которых нет (:not) атрибута alt, необходимо добавить этот атрибут с помощью метода attr. У метода attr два входных параметра: Первый - это параметр, который надо добавить, а Второй - это значение параметра
