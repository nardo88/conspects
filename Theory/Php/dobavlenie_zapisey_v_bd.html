Добавление записей в БД с помощью кода

Добавление записей

Рассмотрим следующий код:

$mysqli->query("INSERT INTO `users` (`login`, `password`, `reg_date`) VALUES ('123', '".md5("123")."', '".time()."')");
  • query - это оператор говорит о том, что добавление записи будет с помощью SQL запроса (в скобках указывается SQL запрос)
  • INSERT INTO - оператор SQL говорит о том что идет добавление записи
  • users - название таблицы куда будет добавляться запись. Обращаем внимание на то что этот атрибут записан в косых одинарных кавычках
  • login, password, reg_date - поля таблицы. так же указываются в косых одинарных кавычках
  • VALUES - какие значения добавляем. (в скобках в одинарных прямых кавычках указываем значения)
  • '".md5("123")."' - здесь пароль будет зашифрован методом md5
  • '".time()."' - дата регистрации будет указана с помощью функции tlme(). эта функция показывает сколько прошло секунд с 1970 года

Редактирование записей

Рассмотрим следующий код:

$mysqli->query("UPDATE `users` SET `reg_date` = '123' WHERE `users`.`id` = 4");
  • UPDATE - изменить
  • `users` - где меняем т.е. в таблице users
  • SET - какое поле
  • `reg_date` - столбец таблицы (в косых одинарных кавычках)
  • '123' - какое значение присвоить (в одинарных прямых кавычках)
  • WHERE - где (буквально) т.е. указываем какую строку меняем. по запросу получается где в таблице users id равен 4. (users и id в косых одинарных кавычках)

Пример где видно как задавать множество условий WHERE:

$mysqli->query("UPDATE `users` SET `reg_date` = '1' WHERE `login` = 'shop' OR (`id` > 4 AND `ID` < 8)");

Иными словами запрос можно прочитать так: изменить в таблице USERS поле REG_DATE (присвоить значение - 1) где LOGIN равен слову SHOP или где ID больше 4 и меньше 8.

Удаление записей

Рассмотрим следующий код:

$mysqli->query("DELETE FROM `users` WHERE `id` > 4 AND `id` < 7");

SQL можно прочитать так: удалить из таблицы users записи где id больше 4 и меньше 7
