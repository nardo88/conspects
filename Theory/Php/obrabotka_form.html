Обработка форм

Создаем форму

<!DOCTYPE html> <html> <head> <title>обработка форм</title> </head> <body> <form name="test" action="check.php" method="post"> <label>Имя:</label><br> <input type="text" name="name"><br> <label>Email:</label><br> <input type="text" name="email"><br> <label>Сообщение</label><br> <textarea name="message" cols="40" rows="10"></textarea><br> <input type="submit" name="done" value="send"> </form> </body> </html>

Результат








Атрибут объекта form action указывает где будут обробатываться данные с формы. В нашем случае они будут обрабатываться в файле check.php.

В атрибуте method указывается метод отправки данных. Существует два метода:

  • get
  • post

Отличие этих методов в основном в том, что при методе get часть данных будет отображаться в адресной строке, когда при post ничего отображаться не будет.

Метод post (так же как и get) создает массив данных, который передала наша форма. Имя этого массива $_POST. Обязательно в верхнем регистре. (в случае с get массив будет называться $_GET). Так как этот массив ассоциативный, то в качестве индексо ячеек будут использоваться имена объектов.

Пример массива:

$_POST["name"]

$_POST["email"]

$_POST["message"]

Если в файле check.php записать такой текст:

<?php print_r($_POST); ?>

то после нажатия на кнопку submit send загрузится файл check.php и на странице отобразится содержимое массива $_POST

Функция isset

Эта функция используется в тех случаях если нам надо что бы часть кода выполнялся только после нажатия кнопки. Если не использовать эту функцию то код будет выполняться при загрузки страницы

Пример

<?php (isset($_POST["done"])){ print_r($_POST); } <html> <head> <title>обработка форм</title> </head> <body> <form name="test" action="check.php" method="post"> <label>Имя:</label><br> <input type="text" name="name"><br> <label>Email:</label><br> <input type="text" name="email"><br> <label>Сообщение</label><br> <textarea name="message" cols="40" rows="10"></textarea><br> <input type="submit" name="done" value="send"> </form> </body> </html>
