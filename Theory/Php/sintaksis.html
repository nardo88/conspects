Синтаксис

<?php

Код php // это комментарий

?>



Переменные

<?php

$myNumber = 45; // объявление переменной типа int

$float = -3.25; // объявление переменной типа float

$string = "Hello"; // объявление переменной типа String

$bool = 45; // логическая переменная

?>



Константы

Объявление константы происходит с помощью оператора define. Этот оператор имеет два параметра:

- имя константы

- значение константы

Пример

define("PI", 3.14);



Арифметические операции

1.

<?php

$x = 3;

$y = 5;

$sum = $x + $y;

?>



2.

<?php

$x = 100;

$x = $x + 10; // т.е. $x станет 110

$x += 10; // тоже самое

?>



Строковые операции

<?php

$str_1 = "первая строка";

$str_2 = "вторая строка";

echo $str_1. $str_2; // точка объединяет строки

?>

echo – выводит текст на экран.



Что бы выводить символы (например " или /), то перед символами ставим бэкслеш ( \ )

Пример

Echo "\ " этот текст в кавычках \" ";
