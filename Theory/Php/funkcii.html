Функции

Пример

function math ($first, $second) { //функция math с входными параметрами first и second

$summa = $first + $second; //переменной summa присваивается сумма входящих параметров

return $summa; // выходной параметр

}

$x = 10;

$y = 12;

$sum = math($x, $y);

В последней строчке мы переменной sum присваиваем значение выходного параметра функции math
