Создание базы данных в mySQL

  1. Открываем phpmyadmin и заходим на вкладку "Базы данных"
  2. В поле "Создать базу данных" пишем имя нашей базы и выбираем сравнение utf8_general_ci, после чего жмем кнопку "Создать"
  3. После того как база создана, заходим в нее и нажимаем создать кнопку "Создать таблицу". До этого надо списать имя нашей таблицы и указать количество столбцов в таблице. Жмем ок.
  4. Откроется мастер создания таблицы. Разберем параметры столбцов:
    • Имя - наименование столбца
    • Тип - тип столбца. самые популярные:
      • INT - целочисленный тип
      • TEXT - текст (подходит для абзацев)
      • VARCHAR - небольшая строка
      • DATE - дата
    • Длина значения - максимальная длина значения в символах
    • По умолчанию - какое значение будет присваиваться столбцу в случае если значение null
    • Сравнение - оставляем пустую строку, либо указываем utf8-general-ci
    • Атрибуты - unsicned - значит что не отрицательное число
    • Null - это значит что столбец может принимать пустое значение
    • Index - PRIMARY - уникальное значение, UNIQUE - уникальное значение внутри совего типа
    • A_I - автоинкремент, значит что значение будет увеличиваться на единицу. Подходит для ID
