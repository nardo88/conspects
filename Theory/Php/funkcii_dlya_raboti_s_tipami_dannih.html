Функции для работы с типами данных

unset() – удаляет переменную, не значение а саму переменную

is_integer() – проверяет тип данных. Если тип integer, то функция возвращает значение true

is_double() – проверяет тип переменной, если число с плавающей точкой то возвращается значение true

is_string() – проверяет тип переменной, если строка то возвращается true

is_bool() – проверяет является ли переменная булевой

is_null() – проверяет не пустая ли переменная. Если пустая то возвращается значение true

is_array() – проверяет является ли переменная массивом. Если да то true

gettupe() – проверяет тип переменной. Возвращает слово описывающее тип переменной (int, string)
