Способы подключения стилей

Способ № 1

Подключение стилей непосредственно в самом теге

пример

<h1 style="color: red"> текст </h1>

Т.е. в теге пишем атрибут style и указываем в кавычках необходимые стили.



Способ № 2

Подключаем стиль в разделе <head>

пример

<head>

<style type= "text/css">

Body {

background-color: #000000;

}

</style>

<head>

В теге style мы указываем тип - "text/css". Далее внутри тега указываем, для какого тега будет применяться эти стили. В нашем случае это тег <body>. В фигурных скобках указываем атрибуты нашего тега, к которому применится стиль и через двоеточие указываем значение атрибута. В нашем случае в стиле записано, что для тега <body> будет применяться стиль, в котором значение атрибута background-color (цвет фона) будет иметь значение #000000.



Способ № 3

Третий способ заключается в создании отдельного CSS файла, в котором будут прописаны стили для тегов. Для того что бы html файл видел наш CSS файл, необходимо в разделе <head> помощью тега <link> указать пусть до нашего фала CSS

пример

<head>

<link rel="stylesheet" type="text/css" href="style.css">

</head>



атрибуты

rel – отношение между текущим документом и файлом CSS

type – тип данных (CSS)

href – путь к файлу CSS



пример содержимого файл CSS

body {

background-color: #000000;

}

h1{

color: red;

}
