Псевдоклассы

Псевдоклассы в CSS указываются в селекторах после двоеточия.

пример

#example: first-letter {}



ПЕРЕЧЕНЬ ПСЕВДОКЛАССОВ:

first-letter – применяет стиль к первой букве строки

first-line – применяет стиль к первой строке абзаца

hover – применяет стиль к объекту при наведении на него мышкой

пример:

.href:hover { color: red; }

Теперь при наведении мыши на ссылку с классом href ссылка будет меня цвет

active – применяем стиль если мы нажимаем мышкой по объекту

visited – псевдокласс только для ссылки, если мы уже прошли по этой ссылке, к ней применится стиль

focus - Псевдокласс :focus определяет стиль для элемента получающего фокус. Например, им может быть текстовое поле формы, в которое устанавливается курсор.

not - Псевдокласс :not задаёт правила стилей для элементов, которые не содержат указанный селектор. пример:

input:not([type="submit"]) { border: 1px solid #ccc; padding: 3px; }

valid - Применяется к полям формы, содержимое которых проходит проверку в браузере на соответствие указанному типу. Например, для type="number" вводится число, а не буквы, для type="email" корректный адрес электронной почты.

invalid - Применяется к полям формы, содержимое которых не соответствует указанному типу. Например, для type="number" должно вводиться число, а не буквы, для type="email" корректный адрес электронной почты.

placeholder-shown - Определяет стиль элемента input или textarea, который в данный момент отображает текст подсказки, заданной атрибутом placeholder. Такая подсказка обычно исчезает при наборе текста.

nth-child - Псевдокласс :nth-child используется для добавления стиля к элементам на основе нумерации в дереве элементов. n/t допустим в блоке div с классом icon у нас несколько span с помощью этого псевдокласса мы можем стилизовать каждый span (.icon span:nth-child(1)) в скобках указываем номер span.
