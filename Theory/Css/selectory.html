Селекторы

Рассмотрим код CSS

Body {

Background-color: red;

}

В нашем случае body – это селектор. То что указано в фигурных скобках называется стили. В стилях указывают свойства и их значения.

Как можно ссылаться к объекту?

Если указать селектор h1, то стили будут применяться ко всем тегам <h1>. В том случае если нам понадобится ссылаться к определенному объекту, для таких случаем есть три способа:

Способ № 1

Обращение по ID

пример

код html:

<h1 id=”header”> текст </h1>

код CSS

#header {

color: red;

}

Id с таким названием можно применять только к одному тегу в документе.

Способ № 2

Обращение по классу

пример

код html

<h1 class=”style”>текст </h1>

<h2 class=”style”>текст </h2>

Код CSS

.style {

Color: red

}

Так же можно написать в CSS так: h3.style { color: red; }
