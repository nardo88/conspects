Magnific Popup

Вызов модального окна

Для работы модального окна от Magnific Popup нам по традиции надо подключить библиотеки:

Файл со стилями от Magnific <link rel="stylesheet" href="css/magnific-popup.css"> Файл JQuery <script src="//ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script> Файл со скриптами от Magnific <script src="js/jquery.magnific-popup.js"></script>

Для подключения файлов со стилями и со скриптами от Magnific, необходимо скачать архив с этими файлами ОТСЮДА

Распаковываем архив и здесь нам нужна папка "dist", внутри этой папки нам понадобятся два файла:

  • magnific-popup.css
  • jquery.magnific-popup.min.js

Эти файлы мы помещаем в папки CSS и js соответственно.

Допустим что нам надо открыть картинку в модальном окне по нажатии на ссылку.

Создаем нашу ссылку. В атрибудет href мы даем путь до картинке и так же присвоим нашей ссылке класс - image_link

После этого нам необходимо инициализировать нашу ссылку, для этого подключаем файл с нашим скриптом и помещаем туда вот такой код:

$(document).ready(function() { $('.image_link').magnificPopup({type:'image'}); });

Вот второй строчке с помощью JQuery мы обращаемся к нашей ссылке по классу image_link

Теперь по нажатии кнопки будет вылезать модальное окно с картинкой.
